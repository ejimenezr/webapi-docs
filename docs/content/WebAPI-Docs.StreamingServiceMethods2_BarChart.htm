TradeStation WebAPI

Streaming Service v2 > BarChart

Summary:¶

Streams barchart information for a particular symbol from a given start date, or bars back. Can also return a barchart for a given time-span.

Details:¶

  • Method: GET
  • Path /stream/barchart/{symbol}/{quantity}/{barInterval}/{startDate}/{endDate}
  • URI Parameters:
    • {symbol} =
    • {barIntervalQuantity} (Must be 1 for interval units Daily, Weekly and Monthly)
    • {barIntervalUnit} = {Second, Minute, Daily, Weekly, Monthly}
    • {startDate} = {m-d-yyyy} - This date value is for time 00:00:00 of that day.
    • {endDate} = {m-d-yyyy} - This date value is for time 00:00:00 of that day.

  • Path /stream/barchart/{symbol}/{quantity}/{barInterval}/{barsBack}/{lastDate}
  • URI Parameters:
    • {symbol} =
    • {barIntervalQuantity} (Must be 1 for interval units Daily, Weekly and Monthly)
    • {barIntervalUnit} = {Second, Minute, Daily, Weekly, Monthly}
    • {barsBack} = Number of bars to go back from the current time or the value provided in lastDate
    • {lastDate} = {m-d-yyyy} - This date value is for time 00:00:00 of that day.

  • Authentication: Requires a valid access token.

Returns:¶

IntradayBarData object

Errors:¶

  • 401 | Unauthorized
  • 5xx | Unknown internal service error. Contact TradeStation.
